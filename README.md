# grow - is a react js project with webpack and babel. it grows from a simple project to SSR 

- Install node and npm

```
    https://nodejs.org/en/
```

- check node and npm version 

```
  node -v
  npm -v
```

- install dependencies using npm i

```
    npm i 
``` 

- start npm run start

```
    npm run start
```

- build npm run build

```
    npm run build
```
